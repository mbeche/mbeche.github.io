Time to start tackling my first sprint by making the Django Project and initiating the fist app. I will fill in the sprint with commands and clues first and then run through the steps.

  1. Create django project
    1. Navigate to chosen directory: C:\Users\Ethan\github\django\IMTwitter
    2. Start project: django-admin startproject IMTwitter
    3. Commit changes to repository
    4. Navigate to project folder: cd IMTwitter
    5. Start server: python manage.py runserver
    6. Bask in the glory of Django 2.0's new project landing page :)
  2. Creating app
    1. I will call this app IMTweet: python manage.py startapp IMTweet
    2. Note: I now have two folders in my project: IMTwitter and IMTweet. IMTwitter is the project folder and IMTweet is the app folder.
    3. Commit changes to repository
  3. Creating the views/url connect for IMTweet index
    1. Create View of index page in IMTweet views.py
    2. Create IMTweet urls.py
    3. Create URLconf for index in IMTweet urls.py
    4. Create URLconf for 'IMTweet/urls.py' in IMTWitter urls.py
    5. Note: I deviated from the tutorial and made the view the main path versus the subpath, can update later if I want a landing page
