A note on styling

I have never used inline styling before - I also have only used internal styling with the style element in the header on the rare occasion of a tutorial where I didn't want to create a linked CSS file. It's always been external for me - the "start with the basics and obey the separations of concerns" camp had won me over.

However, with this application I have started doing inline styling. I have discovered bootstrap is incredibly limited - at least as far as I can tell. If I want to make subtle changes that I can't figure out how to do with bootstrap classes, I just make an inline style. This seems to work out pretty well for me, but I worry I'm getting sloppy. If it gets too out of hand I'll go through and create a CSS file.
