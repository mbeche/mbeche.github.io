CSS and HTML Decoupling with Edx

This is from the first homework assigntment in the course PennX: SD4x - Programming for the Web with JavaScript from Edx. They gave us the the calendar with HTML with inline CSS (version 1). The assignment was the remove the CSS from the HTML file and put it in it's own CSS file (version 2). I decided to take it a step further and move the CSS into SCSS (version 3). I then reformatted the calendar to make one with a different color scheme and a more playful picture I found available for use on deviant art (version 4).

All the code for these versions are available below in trinkets and on my GitHub page. The original version of the calendar looked identical for all three iterations (shown below to the left). My revamped version is shown below on the right.
